Frequently Asked Questions

  • I don't know anything about Coding, is this batch good for me?

    Yes, this course will cover all important concepts from basic till advanced. So, there is no need to know anything about coding beforehand.

  • Do I need to be a Computer Science student to take up this course?

    Anyone, from any background or field of study, can take up this course and learn DSA.

  • Can I do this course with my college classes?

    Yes, it is a self-paced course and can be done along with college classes.

  • In which Language course is?

    The Course is Taught in English

  • How long is the course access?

    The course is accessible for the duration of one year.

  • Is there a course schedule?

    Yes, each lesson will be unlocked on alternative days so that you have ample time to learn core concepts & practice questions.

  • Will there be question practice in the course?

    Yes, the course has 200+ carefully curated questions solved in class.

  • What are Data Structures and Algorithms(DSA)?

    These are tools & techniques that will help us write good code. A data structures is a named location that can be used to store and organize data. An algorithm is a collection of steps to solve a particular problem.

  • Why should I learn Data Structures and Algorithms(DSA)?

    Data Structures and Algorithms are frequently asked in Job interviews of many companies like Microsoft, Google, Amazon, Facebook & many many more. It also help you to write optimized and scalable code. You will also learn to write code that runs faster and requires less storage.

  • Is there a group/community I can join after enrolling?

    Yes, we will inform you about the community to join after the batch starts.

  • Will there be LIVE classes?

    No, this is a recorded self-paced course. There will be some LIVE mentorship sessions regarding interview preparation, resume building, project making etc.

  • What are the class timings?

    The lectures will be in recorded format and uploaded according to schedule(alternate days). The timings for the 7 Live sessions might vary which will be conveyed to students in advance.

  • I just completed 12th and I want to start preparing for my internship/job as a Software Developer, can I take it?

    Yes, you are eligible to enrol as we will cover everything from basics to advanced. It is always better to start as early as possible. It will give you a good head start and ample time to practice.
